# 📱 Web app

> [!TIP]
> [Extensions](https://github.com/VIPnytt/Frekvens/wiki/Extensions) and [modes](https://github.com/VIPnytt/Frekvens/wiki/Modes) should be enabled/disabled using the global [.env](https://github.com/VIPnytt/Frekvens/.env) file in the root directory, as many of the options is shared between the `webapp` and `firmware`.

## 📦 Node.js

Building requires [Node.js](https://nodejs.org). The *LTS* release is recommended. Newer versions will usually work as well.

> [!NOTE]
> If Node.js is not detected on the system, a compatible version will be installed automatically into `.nodeenv`.
