# ⚙️ Firmware

The [secrets.h](https://github.com/VIPnytt/Frekvens/blob/main/firmware/include/config/secrets.h) file is located in `firmware/include/config/secrets.h`.

> [!TIP]
> [Extensions](https://github.com/VIPnytt/Frekvens/wiki/Extensions), [fonts](https://github.com/VIPnytt/Frekvens/wiki/Fonts), and [modes](https://github.com/VIPnytt/Frekvens/wiki/Modes) should be enabled/disabled using the [.env](https://github.com/VIPnytt/Frekvens/blob/main/.env) file in the root directory, as many options is shared between the `firmware` and `webapp`.
